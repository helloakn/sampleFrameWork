# Zote
## Create Project
```
php zote init doc:../project
```
After init prject, file structure will be
```
 []zote-framework
 []project
     .env
     []public
         index.php
         .htaccess
         robots.txt
     []mobile
         User.php
     []migration
         createUser.php
     []controller
         userController.php
     []route.php
```
